Nico Box

Mein Platz. Meine Regeln. Na ja, fast.

Was ist die Nico Box?

Die Nico Box ist genau das, was der Name verspricht: eine Box, die Nico gehört. Kein vorgegebenes Thema, kein Format, kein "das passt hier nicht hin". Einfach ein eigener Bereich auf unserer Gruppenwebsite.

Das ist eigentlich die beste Art eine Website zu lernen – wenn man einfach mal machen kann ohne dass jemand sagt "das geht so nicht". Weil: warum nicht?

Wer ich bin

Ich heisse Nico, bin Schüler und lerne gerade wie das alles funktioniert – HTML, CSS, Git, der ganze Kram. Manchmal macht es Sinn, manchmal frage ich mich, wer sich diese Regeln ausgedacht hat.

Wenn ich ehrlich bin, war ich anfangs skeptisch gegenüber diesem Projekt. Eine Website bauen, in der Schule? Aber es ist interessanter als gedacht. Man sieht relativ schnell Ergebnisse, und das motiviert mehr als eine Seite Theorie.

Was ich gelernt habe

Gruppenarbeit an Code ist komplizierter als gedacht. Nicht wegen den anderen, sondern wegen dem Code selbst. Wenn jeder seinen Teil macht und dann alles zusammengefügt wird, muss man plötzlich verstehen was der andere geschrieben hat.

Git ist genau dafür gemacht. Das kapiert man aber erst, nachdem man es zweimal vermasselt hat.

Was ich wirklich mitgenommen habe: Websites sehen von aussen einfach aus, bis man sie selbst baut. Dann merkt man, wie viel dahintersteckt.
